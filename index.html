Question 4: What is DNS and when it come in picture

DNS (Domain Name System) is a system that translates human-readable domain names (like www.google.com) into machine-readable IP addresses (like 142.250.183.206) required to locate and connect to servers on the internet. When you enter a domain name into a browser, DNS comes into play by resolving the name to its corresponding IP address through a series of queries involving local caches, DNS servers, and authoritative servers. This process happens instantly, enabling seamless access to websites and online services by bridging the gap between human-friendly names and computer-required addresses.

    Question:15 HTML5 vs HTML4

    HTML5 is a significant improvement over HTML4, offering native support for multimedia with audio and video tags, semantic elements like header and section for better structure, and new form controls such as email and date for enhanced usability. It eliminates the need for plugins like Flash by introducing features like the canvas and svg for graphics. HTML5 is mobile-friendly, includes APIs for geolocation and local storage, and simplifies coding with a shorter doctype declaration. Unlike HTML4, which relied heavily on external tools and generic elements, HTML5 is modern, efficient, and designed for responsive web development.

    Question:30 What is the difference between HTML elements and tags?

    In HTML, tags are the markup symbols (like <p> and </p>) used to define the start and end of an element, while an
    element refers to the entire component that includes the opening tag, the content within it, and the closing tag. For
    example, in <p>> This is a paragraph.</p>, <p> and </p> are the tags, and the entire <p> This is a paragraph.</p> is the
    element. So, tags are the syntax used to create elements, and elements are the complete units of content defined by
    those tags.

    Question:37 Does a hyperlink only apply to text?

    Question:41 Explain the layout of HTML?

    The layout of an HTML document typically consists of a <!DOCTYPE html> declaration, followed by the
    <html> element, which wraps all the content on the page. Inside the <html> element, the <head>
    section contains metadata, including the document's title, character encoding, and links to external files like CSS
    or JavaScript. The <body> section holds the visible content of the page, such as text, images, and links.
    Inside the body, there are common structural elements like <header>, <footer>, <nav>,
    <main>, <section>, and <article> that organize and structure the content. The body section is
    where the actual webpage content resides and is styled using CSS and enhanced with interactivity using JavaScript.

    Question:42 What is a marquee?

    A marquee in HTML is a tag used to create a scrolling or moving text or image across the screen. The content inside

    impact user experience. The<marquee> element allows you to control the direction, speed, and behavior of the scrolling content through
    attributes such as direction, scrollamount, behavior, and loop.
